# Land Referee – Constraint-Aware Real Estate Decision Tool

This project demonstrates a referee-style AI system that compares multiple real estate development options under user-defined constraints.

## Key Features
- Constraint-aware reasoning
- Trade-off based explanations
- No single “best” answer
- Built using Node.js and Kiro

## Example Constraints
- Budget
- Risk tolerance
- Time horizon
